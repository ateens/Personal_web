# Personal Web

Static local-first personal operating system prototype.

## Run Locally

```bash
python3 -m http.server 4180 --bind 127.0.0.1
```

Then open:

```text
http://127.0.0.1:4180/
```

## Main Files

- `index.html`
- `styles.css`
- `app.js`
- `manifest.json`
- `service-worker.js`
- `icons/`

`Notion_like/` is a separate earlier static editor prototype kept in the same workspace.
